# Packet Tracer Labs

Welcome to my networking and telecommunications portfolio. This repository serves as a centralized hub for all network labs, simulations, and architectural designs developed throughout my academic and professional growth in Systems Engineering.

> **Repository Description:** 
> Academic and professional portfolio dedicated to designing, configuring, auditing, and troubleshooting network infrastructures using Cisco Packet Tracer.

---

## Repository Structure

To maintain a clean and modular workspace, every lab is organized in its own dedicated directory containing:
* **Source File:** The original Cisco Packet Tracer file (`.pkt`).
* **Network Topology:** High-resolution visual schematic exported directly from the lab.
* **Documentation:** A comprehensive `README.md` guide breaking down addressing tables, CLI configurations, and verification steps.
